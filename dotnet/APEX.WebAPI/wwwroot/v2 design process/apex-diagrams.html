Analyse des Limites du SaaS

Points de vigilance — Agent IA & Architecture

Élevée
Quota Gemini gratuit
60 req/min pour Flash, 2 req/min pour Pro. En cas de pic de charge ou de croissance rapide, le service IA devient indisponible.
✓ Cache des analyses 24h · Limite par plan · Fallback texte générique
Élevée
Dépendance France Travail API
Si l'API France Travail change, est en maintenance ou révoque l'accès, l'ensemble des offres disparaît.
✓ Cache BDD (offres copiées 24h) · Abstraction service · Multi-sources Phase 2 (Arbeitnow, The Muse)
Moyenne
Hallucinations IA
Gemini peut générer des scores ou compétences incorrectes, en particulier pour des métiers peu représentés dans les données d'entraînement.
✓ Disclaimer visible · Température basse (0.1) · Relecture humaine obligatoire
Moyenne
Logos Clearbit
L'API Clearbit peut retourner des logos incorrects, obsolètes ou indisponibles pour certaines entreprises françaises peu connues.
✓ Fallback initiales · Cache 7j côté serveur · Pas de dépôt des logos en BDD
Moyenne
Agent autonome — risques RGPD
La candidature automatique sans validation peut entraîner l'envoi de données personnelles à des tiers sans consentement explicite à chaque action.
✓ Validation obligatoire avant envoi · Log de chaque action · Droit de retrait immédiat
Moyenne
SQL Server Express — 10 Go
La limite de 10 Go peut être atteinte avec des millions d'offres et de CV. La stratégie de rétention RGPD (2 ans) aide naturellement.
✓ Purge automatique · Compression texte · Migration vers SQL Server Standard si dépassement
Faible
Pas de partenariat LinkedIn
Les liens LinkedIn sont des recherches publiques, pas des intégrations officielles. L'API officielle LinkedIn nécessite un partenariat coûteux.
✓ Lien de recherche public suffisant · Mention "Voir sur LinkedIn" sans implication de partenariat
Faible
Micro-entreprise — plafond CA
Au-delà de 93 600€ de CA (prestations de services), le statut micro-entreprise est perdu automatiquement.
✓ Anticiper la création d'une EURL/SASU dès 60 000€ de CA · Suivi mensuel du CA
Faible
Valeurs Gemini — config critique
Les outils Antigravity et Copilot écrasent régulièrement FlashModel/ProModel avec des versions dépréciées.
✓ Valeurs fixées dans appsettings.json · Ne jamais modifier sans validation manuelle · Commentaires d'avertissement
